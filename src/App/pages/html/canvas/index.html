画笔配置

fillStyle / strokeStyle

属性，值应该是一个颜色字符串，分别用于设置图形的填充颜色和图形轮廓的颜色。

lineWidth

属性，值应该是一个数字，用于设置线条的粗细程度。

setLineDash()

方法，此方法接受一个数组，来指定线段与间隙的交替。例如： 

    painter.setLineDash([20, 5]);  // [实线长度, 间隙长度]

font

属性，值应该是一个拼接了文字大小和字体的字符串，用于设置文字大小和字体。例如： 

    painter.font = "100px sans-serif";

textAlign / textBaseline

属性，值应该是一个字符串，分别用于设置文字的水平和垂直对齐方式。

水平方向上，可选的值包括：start、end、left、right、center，默认值是start。

垂直方向上，可选的值包括：top、hanging、middle、alphabetic、ideographic、bottom，默认值是alphabetic。

direction

属性，设置文字的书写方向，可能的值包括：ltr、rtl、inherit，默认值是inherit。

绘制方法

save / restore

一个绘图可能会包含一些属性的设置（比如文字大小等），应用一些变换（即移动、旋转和缩放）等，我们称在某一刻前述的这些的信息为 状态。

你可以通过 save()方法来保存当前状态， 后续操作结束以后，再通过 restore()方法恢复之前保存的状态。

每当save()方法被调用后，当前的状态就被推送到栈中保存。每一次调用restore()方法，上一个保存的状态就从栈中弹出，所有设定都恢复(类似数组的pop())。

fillText / strokeText

分别用于绘制填充文字和轮廓文字，拿fillText举例子，语法如下： 

    painter.fillText(text, x, y);

在指定的 (x,y) 位置填充指定的文本text。

fillRect / strokeRect

分别用于绘制填充矩形和轮廓矩形，，拿fillRect举例子，语法如下： 

    painter.fillRect(x, y, width, height);

下面，我们来对常用的方法进行解释。

beginPath()

一般在生成路径前都会调用这个方法，不然的话，会接着之前的路径生成。

moveTo(x,y)

把画笔移动到指定的坐标，相当于路径的起点。

lineTo(x,y)

把画笔移动到指定的坐标，和moveTo不同的是，此时笔没有离开纸。

closePath()

相当于调用lineTo方法把路径的终点和起点连起来。

stroke() / fill()

路径只是记录了笔的轨迹，要最终可以看见，就需要调用上面的方法告诉画布把记录的路径绘制出来。
